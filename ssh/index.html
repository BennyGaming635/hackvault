Build & Ship: Password Manager

Hackvault is the YSWS track where hack clubbers build a password manager — simple, secure, and friendly. If you ship a fully working password manager, we'll send you a YubiKey for free!

Project constraints

  • Project must be primarily a password manager (desktop, mobile, or web).
  • Include a public repo link (GitHub/GitLab) or upload in the submission form.
  • Solo only - no team submissions.
  • Minimum work time: 5 hours total development time.
  • Double-dipping (using this for another event/project) is allowed.
  • Do not include real passwords in your demo or repo.
  • Bonus: YubiKey support or a plan for hardware-backed secrets!

Need help? Check out the guide in #hackvault on Slack, where we can help you

Questions? Reach out to @BennyGaming635 or just ask in #Hackvault.
